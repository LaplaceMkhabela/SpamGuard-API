POST/predict

Classify Message

Performs a multi-layered analysis on a text string. Combines Scikit-Learn spam model + phishing heuristics + profanity filter.

Request Body (JSON)

Field Type Description
message string The raw text to analyze. (Required)

Response Object

Field Type Description
prediction string SPAM or HAM
spam_probability float Confidence (0-1) from ML model.
phishing_probability float Heuristic score based on URLs/banking keywords.
profanity_detected boolean True if offensive words matched.
language string ISO language code (e.g., en-US).

cURL Example

curl -X POST https://api.spamguard.dev/predict \
  -H "Content-Type: application/json" \
  -d '{
    "message": "URGENT: Verify your account at bit.ly/fake-link"
  }'

Sample Response

{
  "prediction": "SPAM",
  "spam_probability": 0.9214,
  "phishing_probability": 0.6500,
  "profanity_detected": false,
  "language": "en-US"
}
